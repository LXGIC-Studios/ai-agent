# AI Agent

AI agent framework scaffolding. ReAct agents with tool use.

## Quick Start

```bash
npx ai-agent demo "What is 25 * 4?"
```

## What It Does

- Create ReAct agents
- Define custom tools
- Multi-step reasoning
- Type-safe with Zod

## Usage

```bash
# Demo
npx ai-agent demo "task"

# Generate files
npx ai-agent init
```
